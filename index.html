about

seiya imoto

静岡県富士市在住のWebコーダー志望、せいやです。2022年から2024年にかけてアイルランドに留学し、世界各国から集まる人たちと交流する中で、Web制作やプログラミングの可能性に惹かれるようになりました。帰国後はその思いを形にするため、2024年6月よりコーディングの学習をスタートし、挑戦を続けています。趣味： 英会話、スケートボード、海外旅行、コーディング、映画鑑賞

vision

目指すコーダー像

コーディングに加え、デザインやWordPressなどのスキルも磨き、幅広い制作に対応できる、コーダーを目指しています。ゆくゆくは、デザインから実装まで一貫して担える制作者として成長したいです。

入社後の成長目標

  • ３ヶ月後

    制作フローやコーディングルールを理解し、WordPress業務を含めた日々の制作業務に慣れる。修正指示にも対応しながら、安定したコーディングができる状態を目指します。

  • ６ヶ月後

    全体的な業務に慣れ、小規模なページや更新案件であれば1人で任せていただけるコーダになる。また並行してWEBデザインの学習にも力を入れていきます。

  • 1年後

    デザイン意図を理解した実装ができ、難易度の高い案件にも対応できるWEBデザイナー兼コーダーとして成長する。また積極的に改善提案をするなど、能動的に制作に関わっていきます。

skills

  • コーディング

    デザインカンプをもとに、HTML / CSS / JavaScript を使用して正確かつ再現度の高いコーディングが可能です。 レスポンシブ対応やCSS設計を意識し、保守性・再利用性の高いコードを書くことを心がけています。

  • アニメーション

    JavaScriptライブラリの GSAPを使用し、スクロールアニメーションやインタラクティブな動きを実装できます。ユーザー体験を高めるために自然で洗練されたアニメーションを追求しています。

  • デザインツール

    Figmaを使用して、デザイン情報の確認、画像の書き出しなどの基本操作が可能です。自身のポートフォリオサイトは、Figmaで自らデザインしました。

  • 今後の学習領域

    現在Wordpressを学習しています。サイトの構築から運用まで対応できるよう、実践を意識した学習をすすめています。
